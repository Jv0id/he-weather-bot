# 和风天气预报机器人 HeWeatherBot

## 一、背景

基于和风天气 & telegram机器人的天气预报小棉袄。根据用户地区，查询实时天气，或每天固定时间自动播报天气预报。

## 二、交互

```
weather - 查看今日天气
update - 更新地址️(城市/定位)
subscribe - 开启订阅
unsubscribe - 取消订阅
```
